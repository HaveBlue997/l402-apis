How It Works

The L402 protocol in 5 steps — no signup, no API keys, just Lightning

1

Discover

Agent discovers the API endpoint and available services via OpenAPI spec.

2

Request

Agent makes a request → receives HTTP 402 Payment Required with a Lightning invoice.

3

Pay

Agent pays the Lightning invoice (50–100 sats, ~$0.0005–$0.001). Instant settlement.

4

Authenticate

Agent receives an L402 macaroon — a cryptographic access token tied to the payment.

5

Access

Agent re-sends the request with the macaroon → receives the data. Done.

Full agent integration flow 
# Step 1-2: Make initial request
$ curl -si https://nautdev.com/api/v1/sanctions/check?name=John+Doe
# → HTTP/1.1 402 Payment Required
# → Www-Authenticate: L402 macaroon="...", invoice="lnbc1000n1p..."

# Step 3: Pay the invoice with any Lightning wallet/node
$ lncli payinvoice lnbc1000n1p...
# → Payment preimage: abc123def456...

# Step 4-5: Re-send with L402 credentials
$ curl -H "Authorization: L402 <macaroon>:<preimage>" \
  https://nautdev.com/api/v1/sanctions/check?name=John+Doe
# → HTTP/1.1 200 OK
# → { "matches": [], "status": "clear" }

Python agent integration 
import httpx
from l402 import L402Client

# L402-aware HTTP client handles 402 → pay → retry automatically
client = L402Client(
    lnd_host="localhost:10009",
    macaroon_path="~/.lnd/data/chain/bitcoin/mainnet/admin.macaroon"
)

# Just make the request — payment is handled transparently
response = client.get(
    "https://nautdev.com/api/v1/sanctions/check",
    params={"name": "John Doe", "country": "US"}
)

print(response.json())
# → {"matches": [], "checked": 18706, "status": "clear"}
